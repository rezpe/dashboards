Bar Chart expects the labels to be an array of strings (ex: ["Pear","Banana", ... ]) and the data to be an array of arrays (ex: [[0,1,2, ...], [3,4,5, ...], ...])
Line Chart expects the labels to be an array of strings (ex: ["Pear","Banana", ... ]) and the data to be an array of arrays (ex: [[0,1,2, ...], [3,4,5, ...], ...])
Pie Chart expects the labels to be an array of strings (ex: ["Pear","Banana", ... ]) and the data to be an array (ex: [0,1,2, ...] )
Doughnut Chart expects the labels to be an array of strings (ex: ["Pear","Banana", ... ]) and the data to be an array (ex: [0,1,2, ...])
Radar Chart expects the labels to be an array of strings (ex: ["Pear","Banana", ... ]) and the data to be an array of arrays (ex: [[0,1,2, ...], [3,4,5, ...], ...])
Table expects the labels to be an array of strings (ex: ["Pear","Banana", ... ]) and the data to be an array of arrays (ex: [[0,1,2, ...], [3,4,5, ...], ...])
Google Map Chart expects the center to be a place that can be geolocated. The scale is a the zoom level (integer between 1-10). The points to be an array of objects with id, name, latitude and longitude attribute (ex: [{ id: "City A", name:"City A", latitude: 2.3, longitude:3.4 }, ... ]). The routes need to be an array of objects with attribute origin (string), destination (string) path which is an array of objects with latitude and longitude attributes (ex: [{ origin: "Paris", destination: "Madrid", path:[{longitude:2.3,latitude:3.4},{longitude:2.3,latitude:3.4}]}, ...])
